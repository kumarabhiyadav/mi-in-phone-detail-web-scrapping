# Extract Data of mobile phone from https://www.mi.com/in/


### Install Dependencies
```
$ pip install -r requirements.txt
```

### Run main.py

```
python main.py
```

## How to use 

1.  append URL to URLS array
2.  run main.py 
3. get output.json about phone details
